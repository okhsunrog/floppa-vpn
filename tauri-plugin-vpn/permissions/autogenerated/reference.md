## Default Permission

Default permissions for VPN plugin

#### This default permission set includes the following:

- `allow-prepare-vpn`
- `allow-start-vpn`
- `allow-stop-vpn`
- `allow-get-vpn-status`
- `allow-get-installed-apps`
- `allow-protect-socket`
- `allow-get-safe-area-insets`
- `allow-register-listener`
